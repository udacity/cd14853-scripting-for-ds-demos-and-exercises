Tasks
1) Create a Script File
Create a shell script named:
scan_events.sh
2) Add Script Contents
The script must:
- Print a short header message
- Print the total number of lines in data/events.log
- Print the number of lines containing:
    - ERROR
    - WARN
You may assume the script is run from the project root.

3) Make the Script Executable
Update permissions so the script can be executed directly.

4) Run the Script
Run the script using one of the supported methods:
./scan_events.sh
bash scan_events.sh
